# NOTICE

## Why this repo is public but not open-source

PreSold is a working business. The source is published so portfolio readers,
prospective collaborators, and curious developers can see how it is built —
the planning discipline, the architecture, the trade-offs, the per-session
log of decisions.

It is **not** open-source. There is no licence to copy, fork, modify, or use
this code in a product of your own. The full terms are in [LICENSE](./LICENSE).

If you want to use any of this for something real, contact me and we can
talk.

## AI assistance

This project is built with the help of [Claude Code](https://claude.com/claude-code)
(Anthropic's Claude Opus 4.7). Most commits include a `Co-Authored-By` line
naming the model. This is transparency, not deflection — every decision,
every line that landed, and every bug is mine. The model is a collaborator,
not a co-founder.

You can see exactly how AI is used by reading the PR descriptions: each one
is a session log that records the goal, the decisions, and the files
touched. Nothing is hidden.
